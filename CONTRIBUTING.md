# 贡献指南

感谢你愿意改进奶蛙 Codex Pet。

## 可以贡献什么

- 修正中文说明、安装步骤或兼容性问题。
- 改进 macOS、Linux 或 Windows 安装脚本。
- 报告不同 Codex 版本中的触发行为。
- 在保持角色一致性的前提下改进动画连贯性。

## 提交问题时请附上

- 操作系统与版本。
- Codex 桌面版版本。
- 安装方式：脚本、手动复制或 Codex 自动安装提示词。
- `pet.json` 是否能被发现。
- 问题对应的动作、触发方式和截图或录屏。

请不要提交个人路径、账号信息、令牌、缓存、日志中的敏感内容或从网络收集的原始参考图。

## 修改图集时必须保持

- 图集格式：Codex 自定义宠物 v2。
- 整体尺寸：`1536 × 2288`。
- 网格：`8 × 11`。
- 单帧尺寸：`192 × 208`。
- `pet.json` 中的 `spriteVersionNumber` 为 `2`。
- 透明背景，无文字、场景、阴影、漂浮特效或跨格内容。
- 原有状态顺序与帧数约定。

修改 `spritesheet.webp` 后，请同步更新：

- `assets/` 中受影响的预览文件。
- `SHA256SUMS` 中的校验值。
- README 中受影响的动作说明。

## Pull Request 建议

1. 每个 Pull Request 尽量只解决一个问题。
2. 在说明中写清改了什么、为什么改以及如何验证。
3. 图像改动请提供修改前后的正常尺寸预览。
4. 不要把生成过程中的大体积临时文件、原始网络参考图、缓存或本机安装目录提交进仓库。
